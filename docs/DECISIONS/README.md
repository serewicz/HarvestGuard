# Architecture Decision Records

This directory contains HarvestGuard architecture decision records (ADRs).
ADRs capture decisions that shape product direction, contribution review, and
implementation boundaries.

- [ADR-005: Evidence versus inference](ADR-005-evidence-versus-inference.md)

## Status Labels

- `Accepted`: this decision is current product direction.
- `Superseded`: this decision has been replaced by a later ADR.
- `Proposed`: this decision is under discussion and should not guide
  implementation until accepted.

Historical notes from `docs/decision-log.md` have been preserved here as formal
ADRs where they affect architecture or contribution direction.
